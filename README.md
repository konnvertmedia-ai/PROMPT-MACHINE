# Konnvert Prompt Machine

Generate production-ready prompts for AI video and image tools instantly.

Built for DTC brands, performance marketers, and AI video creators who need
precise, copy-paste-ready prompts without the guesswork.

## Tools Supported

- Kling 3.0 — AI video generation
- Nano Banana Pro — AI image generation
- HeyGen — AI avatar video
- Higgsfield — multi-model AI video
- Grok Aurora — photorealistic image generation
- Flow (Sora) — long-form AI video

## Features

- Strict schema per tool with exact parameter names
- Scene-by-scene breakdowns for video ads
- Consistency lock, negative prompts, safe zone rules
- Clipboard history saved to your browser
- UGC, cinematic, product, Pixar, claymation, interview, podcast formats
